# 🤖 AI Loan Assistant Chatbot

An AI-powered chatbot designed to assist users with **loan-related queries, eligibility checks, and financial guidance** in a conversational manner.

This is an upgraded and production-focused version of the original project, enhanced with better structure, scalability, and real-world usability in mind.

---

## 🌟 Overview

The AI Loan Assistant Chatbot leverages **Natural Language Processing (NLP)** and **Machine Learning techniques** to understand user queries and provide relevant responses.

It simulates how modern financial institutions can automate **customer support systems**, improving efficiency and user experience.

---

## ✨ Key Features

* 💬 Interactive chatbot for loan-related queries
* 🧠 NLP-based intent recognition system
* ⚡ Fast and responsive answers to common financial questions
* 🏗️ Scalable and modular architecture
* 🔧 Easy to extend with new intents and features
* 🌐 Web-based chatbot interface

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/ShuklaChinmay/AI-Loan-Assisstant-Agent-.git
```

### 2️⃣ Navigate to the project directory

```bash
cd AI-Loan-Assistant-Chatbot
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the application

```bash
python app.py
```

### 5️⃣ Open in browser

```
http://localhost:5000
```
